New package: bundler plugin built on `unplugin`. Resolves a virtual module (`virtual:fn-schema/bundle`) to the extracted JSON Schema bundle and watches source files for HMR in Vite dev mode. Subpath exports cover Vite, webpack, Rollup, esbuild, Rspack, Rolldown, and Farm.
